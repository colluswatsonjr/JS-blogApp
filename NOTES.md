# Overview (what are we building?)
Blog app, where users can create and edit blogs.

## Features (what can it do?)
-Render all currently saved blogs.
-Create new blogs.
-Saves new blogs.
-Edit creates blogs.
-Delete created blogs.
-View deleted blogs.

## MVP
-Render Blogs (DOMContentLoaded)
-Create Blog (submit)
-Edit Blog (click)
-Delete Blog (click)
-Select (change)
